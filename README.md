This is a collection of python scripts.

* new_wifi_bt_pcap_analyzer script analizes .pcap files from wireshark, tcpdump, and tshark as well as ESP32 devices and nRF Bluetooth Dongles

* strng_arm is a password generator using the secrets python library to produce strong passwords

* qr_generator produces QR code images based off of what you input into the prompt

* password_checker checks your password to tell you if its strong or weak
